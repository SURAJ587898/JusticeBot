JusticeBot - AI Legal Assistant

JusticeBot is a responsive, web-based legal assistant chatbot powered by Google's Gemini API. It helps users understand legal concepts, procedures, and rights through an intuitive chat interface.

Since this is a single-file application, setup is extremely simple.

Clone the Repository (or download the files):

git clone [https://github.com/yourusername/justice-bot.git](https://github.com/yourusername/justice-bot.git)


Configure API Key:

Open index.html in any text editor (Notepad, VS Code, etc.).

Locate the script section at the bottom of the file.

Find the line: const apiKey = "YOUR_API_KEY_HERE";

Ensure your valid Gemini API key is pasted inside the quotes.

Run:

Simply double-click index.html to open it in your default web browser.

üîë Getting an API Key

To make the bot work, you need a free API key from Google:

Go to Google AI Studio.

Click Create API key.

Copy the key and paste it into your index.html file.
